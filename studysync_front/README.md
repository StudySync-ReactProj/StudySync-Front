# StudySync — Personal Study Management App

StudySync designed to help students efficiently manage their learning workflow.
Users can track weekly progress, schedule study sessions using calendar synchronization, and organize course tasks — all in one place.

## Required Pages (3 Main Components)

| Page               | Component    | Description                                                            |
| ------------------ | ------------ | ---------------------------------------------------------------------- |
| **Login Page**     | `Login`      | Allows users to sign in (form with validation)                         |
| **Dashboard Page** | `Dashboard`  | Will displays study progress, streaks, and recent tasks with charts    |
| **Tasks Page(API)**| `TasksPage`  | Shows a list of enrolled courses and related upcoming assignments      |


**Run using the commend: npm run dev**
